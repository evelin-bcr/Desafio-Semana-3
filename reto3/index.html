Reto #3 JS

Contexto: Generador interactivo de tablas de multiplicar.

Funcionalidad:

  • El programa debe mostrar la tabla de multiplicar (del 1 al 10) de un número ingresado por el usuario

    • Preguntar si desea ver otra tabla

  • Si responde "sí": repetir el proceso
  • Si responde "no": finalizar el programa
